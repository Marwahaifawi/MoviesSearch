
# Movie Search Website

This is a basic movie search website built with React JS. It allows users to search for movies and get information about them, using an external API.

## Installation

To get started with the installation, follow these steps:

1. Clone the repository to your local machine.
2. Open the terminal and navigate to the root directory of the project.
3. Run npm install to install all the dependencies.

    
## Usage

To use the Movies Search Website, follow these steps:

Navigate to the root directory of the project.

Run npm start to start the development server.

Open your web browser and navigate to http://localhost:3000 to view the website.

Alternatively, you can also visit the website on GitHub Pages by clicking on the following link: https://marwahaifawi.github.io/MoviesSearch/
## Features

- Search for movies by title.
- View a list of matching movies, with poster images and basic information.
- Hover on a movie to view more details, including a larger poster image and additional information such as release year.
- Mobile-responsive design for ease of use on smaller screens.
